Title: GenAI FAQs & Ethical Use of AI

Short overview:
This project is for my pre final exam that explain the GenAI FAQs,. Ethical Use of AI , and Analogy on the use of AI in Dev Work. It took me days to figure it out how to do this project and manage to complete it.

Instructions on how to run the project:

To run this project first you need to have a 4 index html then you have a asset file contains css file and image file. Inside the css file their must be have a index css and in image file their must be 3 image that have a correct name for the code otherwise no background. Then all of this store in one file.

Frameworks/libraries used:

I use vscode to do this project.
